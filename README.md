# 🚗 Guia para o Primeiro Carro

## Sobre o projeto

O **Guia para o Primeiro Carro** é um site desenvolvido como trabalho da disciplina de **Educação Digital: Programação e IA**, com o objetivo de ajudar pessoas que estão comprando seu primeiro veículo.

O projeto reúne informações essenciais sobre planejamento financeiro, escolha do carro ideal, custos de manutenção e respostas para as dúvidas mais comuns dos motoristas iniciantes, utilizando uma linguagem simples e acessível.

---

## Objetivos

- Informar sobre os custos reais de possuir um veículo.
- Apresentar modelos de carros recomendados para iniciantes.
- Explicar despesas como IPVA, seguro, licenciamento e combustível.
- Reunir perguntas frequentes (FAQ) sobre compra e manutenção de veículos.
- Auxiliar na tomada de decisões para evitar prejuízos e golpes.

---

## Funcionalidades

- 📖 Guia para compra do primeiro carro.
- 💰 Informações sobre custos de manutenção.
- 🚘 Lista de carros recomendados.
- ❓ Seção de perguntas frequentes (FAQ).
- 📱 Interface simples e de fácil navegação.
